Topic: Single lines, no separation

	A - aaa aaa aaa.
	B - bbb bbb bbb.
	C - ccc ccc ccc.


Topic: Single lines, separation

	A - aaa aaa aaa.

	B - bbb bbb bbb.

	C - ccc ccc ccc.


Topic: Multiple lines, no indent

	A - aaa aaa aaa
	aaa aaa aaa.
	B - bbb bbb bbb
	bbb bbb bbb.

	zzz zzz zzz.


Topic: Multiple lines, indent, and multiple paragraphs

	A - aaa aaa aaa
	  aaa aaa aaa.
	B - bbb bbb bbb
	  bbb bbb bbb.

	  bbb bbb bbb
	  bbb bbb bbb.

	zzz zzz zzz.


Topic: Blank line above list is required

	zzz zzz zzz.
	A - aaa aaa aaa.


Topic: Traps

	-i - This is a definition list, not a bullet list.


Topic: Quotes and braces

	This should work - as a definition list.

	This should not "work - as a" definition list.

	This should not (work - as a) definition list.

	This should not "[work] - [as a]" definition list.

	This should [work] - "as a" definition list.

	This should [work - as a" definition list.

	This should "work - as a definition list
	since the close" is on a different line.

	This should [work - as a definition list
	since the close] is on a different line.
